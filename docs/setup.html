Setup

Install Xel

Run from console:

npm install xel

Link Xel

Add to the <head>:

<script type="module" href="node_modules/xel/xel.min.js"></script>

Link Xel theme

Add to the <head> one of the following:

Galaxy theme:
<link rel="stylesheet" href="node_modules/xel/stylesheets/galaxy.theme.css">
MacOS theme:
<link rel="stylesheet" href="node_modules/xel/stylesheets/macos.theme.css">
Material theme:
<link rel="stylesheet" href="node_modules/xel/stylesheets/material.theme.css">
Vanilla theme:
<link rel="stylesheet" href="node_modules/xel/stylesheets/vanilla.theme.css">

Link fonts

Some themes expect specific fonts to be linked or embedded by your app:

Galaxy, Material and Vanilla themes:
Roboto and Roboto Mono. For non-latin scripts, also include Noto Sans. Use Google Fonts service to generate necessary code.
MacOS theme:
Relies on fonts provided by the operating system, you don't have to link anything.

FAQ

Q: What are the supported platforms?

Xel works only in environments that support latest Web Platform APIs such as Custom Elements v1, Shadow DOM v1, SVG v2, Pointer Events, CSS variables and ES2017:

  • Atom Electron >= 1.7
  • NW.js >= 0.23
  • Chrome >= 58
  • Chromium >= 58
  • Opera >= 45
  • Android WebView >= 58

Q: Why is my app broken after the latest Xel update?

This project is still in an alpha stage and the API might change occasionally. The first stable version is scheduled for release in 2018 or later.

Q: Can I use it with React, Polymer, Bootstrap or some other framework?

Yes, Xel is framework-agnostic.
